Vuls: VULnerability Scanner
https://github.com/future-architect/vuls

Vuls とは?#

Vulsは、フューチャー株式会社がOSSとして公開しているGolang製の脆弱性スキャンツールです。2022年1月現在、GitHub上で約9000件のスターがついています。

今回はデスクトップPCで使っているOS (Arch) と各種ソフトウェアの脆弱性を確認する目的で、Docker上で実行してみました。

導入方法#

まずは Arch Wiki に従ってDockerをインストールします。

pacman -S docker
systemctl enable --now docker
sudo docker info

つぎに、公式ドキュメントに従ってインストールを行います。

cd 
git clone https://github.com/vulsio/vulsctl.git
cd vulsctl
git pull
cd docker
pwd #/home/eb1ta1/vulsctl/docker

そして脆弱性情報のデータベースを取得します。

# Fetch NVD
docker run --rm -it \
    -v $PWD:/go-cve-dictionary \
    -v $PWD/go-cve-dictionary-log:/var/log/go-cve-dictionary \
    vuls/go-cve-dictionary fetch nvd
# Fetch OVAL
docker run --rm -it \
    -v $PWD:/goval-dictionary \
    -v $PWD/goval-dictionary-log:/var/log/goval-dictionary \
    vuls/goval-dictionary fetch redhat 5 6 7 8

# Fetch gost
docker run --rm -i \
    -v $PWD:/gost \
    -v $PWD/gost-log:/var/log/gost \
    vuls/gost fetch redhat

# Fetch go-exploitdb
docker run --rm -i \
    -v $PWD:/go-exploitdb \
    -v $PWD/go-exploitdb-log:/var/log/go-exploitdb \
    vuls/go-exploitdb fetch exploitdb

# Fetch go-msfdb
docker run --rm -i \
    -v $PWD:/go-msfdb \
    -v $PWD/go-msfdb-log:/var/log/go-msfdb \
    vuls/go-msfdb fetch msfdb

# Fetch go-kev
docker run --rm -i \
    -v $PWD:/go-kev \
    -v $PWD/go-kev-log:/var/log/go-kev \
    vuls/go-kev fetch kevuln
