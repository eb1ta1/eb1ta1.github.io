Manjaro Linux (GNOME) の初期設定ログ

GNOME版の Manjaro Linux をインストール&環境構築した際の個人的な備忘録です。

はじめに#

Manjaro Linux を選んだ理由#

  • Arch Linux 試してみたい
  • だけど素の Arch Linux はチョット怖い

→ Manjaro Linux …!? ええやん。

デスクトップ環境の選択#

デスクトップ環境には小学生の頃から慣れ親しんでいるGNOMEを選びました。本当はxfceを使いたかったのですが、HiDPI周りの設定がうまく行かず諦めました。

インストール#

ISOを配布している公式サイトにアクセスし、ダウンロードしてUSBに書き込みます。

パーティショニング#

PartitionFile SystemMount PointSize
/dev/nvme0n1p1fat32/boot/efi300MB
/dev/nvme0n1p2ext4/465GB

のように設定しました。

初期設定#

GPUドライバの導入#

sudo mhwd -a pci nonfree 0300

HiDPIの設定#

gsettings set org.gnome.desktop.interface text-scaling-factor 1.35

yayの導入#

pacman -S base-devel
git clone https://aur.archlinux.org/yay-bin.git yay-bin
cd yay-bin
makepkg -si

capsをctrlに変更#

gsettings set org.gnome.desktop.input-sources xkb-options "['ctrl:nocaps']"

マウスカーソルの見た目#

GNOMEのデフォルトに戻します。

gsettings set  org.gnome.desktop.interface cursor-theme 'Adwaita'

Noto Sans を導入#

システムフォントを Noto Sans JP に変更します。

yay -S noto-fonts-cjk noto-fonts-emoji
gsettings set org.gnome.desktop.interface font-name 'Noto Sans CJK JP Medium 11'
gsettings set org.gnome.desktop.interface document-font-name 'Noto Sans CJK JP Regular 11'
gsettings set org.gnome.desktop.interface monospace-font-name 'Source Code Pro 11'
gsettings set org.gnome.desktop.wm.preferences titlebar-font 'Noto Sans CJK JP Regular 11'

fcitx5 (+mozc) の導入#

yay -S fcitx5-im fcitx5-mozc
sudo sh -c 'echo -e "GTK_IM_MODULE=fcitx\nQT_IM_MODULE=fcitx\nXMODIFIERS=@im=fcitx" >> /etc/environment'
# GTK_IM_MODULE DEFAULT=fcitx
# QT_IM_MODULE  DEFAULT=fcitx
# XMODIFIERS    DEFAULT=\@im=fcitx

gitのセットアップ#

MAIL='johndoe@example.com'
NAME='John Doe'
yay -S git github-cli
ssh-keygen -t ed25519 -C $MAIL
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
gh auth login
gh ssh-key add ~/.ssh/id_ed25519.pub
git config --global user.name $NAME
git config --global user.email $MAIL

Chromeのネイティブダークモードを有効化#

/opt/google/chrome/google-chrome の最終行を以下に変更します。

exec -a "$0" "$HERE/chrome" "--enable-features=WebUIDarkMode" "--force-dark-mode" "$@"

GNOME Shell のテーマを変更#

# User Themes extension を有効化
gnome-extensions enable user-theme@gnome-shell-extensions.gcampax.github.com

Gnome Tweaks >> Appearance >> Shell から好きなテーマを選べます。

マウスの感度を向上 (Logitech製デバイスのみ)#

yay -S solaar
solaar # 起動 & 設定を変更

# ログイン時に起動するようautostartディレクトリにファイルを作成
touch ~/.config/autostart/solaar-launch.desktop                    
echo "
[Desktop Entry]
Type=Application
Exec=solaar --w hide
Hidden=false
X-GNOME-Autostart-enabled=true
Name=solaar-launch
Comment=solaar-launch
" >> ~/.config/autostart/solaar-launch.desktop
