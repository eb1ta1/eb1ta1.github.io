ブログを始めました。

“eb1note” の由来#

「海老で鯛を釣る」という諺からとった「エビ」と、英語で文書やメモなどの意味を持つ「ノート」を組み合わせた造語です。「ebinote」だとに読みにくいので、あえて「eb1note」にしています。

記事のカテゴリー#

  • #FM → ポッドキャスト的な感覚で雑多に書きます。
  • #Poem → 「お気持ち表明」が主体の記事です。
  • #Idea → 色々と気づいたことなどを書きます。

サイト構成#

Hugo + GitHub Pages で運用しています。近日中に独自ドメインを取得する予定です。はてなブログにするか迷いましたが、VSCodeでサクサクと書きたかったのでHugoを選びました。

デザイン面でのこだわり#

モノトーンを基調に、文章の読みやすさを重視したデザインです…とイキりたいところですが、実際はhugo-PaperModというHugoの人気テーマをほぼそのまま使っています。
