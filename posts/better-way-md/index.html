はじめに#

VSCode で Markdown を書く際におすすめな拡張機能をまとめた記事です。

拡張機能#

Markdown Preview Github Styling#

https://marketplace.visualstudio.com/items?itemName=bierner.markdown-preview-github-styles

Markdown のプレビュー表示が GitHub 風になります。

Markdown Memo#

https://marketplace.visualstudio.com/items?itemName=svsool.markdown-memo

[[link-to-page]]のような WikiLink 形式のリンク表示や、ファイル名の変更リファクタリング、マウスホバーによるテキスト/画像のプレビューなどを可能になります。

Paste Image#

https://marketplace.visualstudio.com/items?itemName=mushan.vscode-paste-image

Cmd+Alt+Vを押すことで、クリップボード内の画像を直接貼り付けることができるようになります。

Markdown All in One#

https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one

キーボードショートカットが導入されたり、リストの入力補完が効くようになったり、目次が作成されたりと、色々便利な機能を使えるようになります。

markdownlint#

https://marketplace.visualstudio.com/items?itemName=DavidAnson.vscode-markdownlint

Markdown 用の linter です。settings.jsonを以下のように設定することで、Markdown 用のフォーマッターとして設定できます。

"[markdown]": {
  "editor.defaultFormatter": "DavidAnson.vscode-markdownlint"}

vscode-textlint#

https://marketplace.visualstudio.com/items?itemName=taichi.vscode-textlint

テキスト用の linter です。npm を通してルールを導入し、.textlintrcに記述することで有効化できます。私はsettings.jsonと.textlintrcをそれぞれ以下のように設定しています。

"textlint.configPath": "$HOME/.textlintrc",
"textlint.autoFixOnSave": true

{
  "filters": {},
  "rules": {
    "preset-ja-technical-writing": true,
    "spellcheck-tech-word": true,
    "preset-ja-spacing": {
      "ja-space-between-half-and-full-width": {
        "space": "always"
      }
    }
  }
}

さいごに#

VSCode の拡張機能の充実ぶりとコミュニティの活発さは本当に凄いなと改めて思いました。
